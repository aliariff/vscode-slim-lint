## Requirements

```
gem install slim_lint
```

## Extension Settings

* `slimLint.executablePath`: Path to slim-lint executable (default: slim-lint)
* `slimLint.configurationPath`: Path to slim-lint configuration file (default: .slim-lint.yml)
